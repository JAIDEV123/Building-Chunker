Code Structure


The NLP lab is built using Flask and the templating engine jinja. The primary reason flask is used is to facilitate future expansion towards interactive content while being lightweight enough for quick and easy development. The logic behind most experiments are implemented in Python 


-Building Chunker
	-- > main.py is reponsible for the flask setup and routing
	-- > static/ contains all static parts such as the css, js and more
	-- > templates contains all the HTML files. These can only be viewed using localhost as rendering is done only when flask app is initialised.
